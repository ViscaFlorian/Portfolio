SAE 5.02 : Pilotage de Projet

Présentation

Réalisation d'une application de partage de voyage en Python (Méthode Agile).

Explication & Réalisation

Dans le cadre de cette SAÉ, nous avons travaillé en grande équipe (7-8 étudiants) pour concevoir une application complète. L'objectif était de simuler un environnement d'entreprise avec une répartition stricte des rôles (Chef de projet, Scrum Master, Responsable Tests, Responsable Documentation).

Environnement Technique & Outils

  • Langage : Python (Gestion des dépendances via Conda).
  • Collaboration : Trello (Suivi des tâches), GitHub (Versionnage).
  • Qualité (QA) : Tests unitaires avec PyTest et documentation automatique via Sphinx (Norme PEP8).
  • Déploiement : Conteneurisation de l'application avec Docker.

Fonctionnalités de l'application

L'application permet la création de voyages (étapes, photos, textes), l'affichage dynamique sur une carte interactive, ainsi qu'un aspect social avec gestion d'amis et options de partage (public/privé).

UE PROGRAMMER

Créer des outils et applications informatiques pour les R&T. Le but est d'acquérir des compétences dans les systèmes informatiques, le codage d'algorithmes, l'architecture Web, et les outils collaboratifs.

Composantes Essentielles

  • CE3.01 | Être à l’écoute des besoins du client
  • CE3.02 | Documenter le travail réalisé
  • CE3.03 | Utiliser les outils numériques à bon escient
  • CE3.04 | Choisir les outils de développement adaptés
  • CE3.05 | Intégrer les problématiques de sécurité

Apprentissages Critiques (Niveaux)

✅ BUT 1

  • Utiliser un système informatique et ses outils
  • Lire, exécuter, corriger et modifier un programme
  • Traduire un algorithme
  • Connaître l’architecture d’un site Web
  • Choisir les mécanismes de gestion de données
  • S’intégrer dans le travail collaboratif

✅ BUT 2

  • Automatiser l’administration système avec des scripts
  • Développer une application (Web/Mobile)
  • Utiliser un protocole réseau (Client/Serveur)
  • Administrer un système de gestion de données

❌ BUT 3 (En cours)

  • Élaborer le cahier des charges d’une application
  • Mettre en place un environnement collaboratif
  • Déployer et maintenir une solution informatique
  • Sécuriser l'environnement numérique d'une application

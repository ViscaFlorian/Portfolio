SAE 1.05 : Python & Scapy

Présentation

Nous avons dû réaliser deux scripts pour manipuler des données réseaux.

Explication & Réalisation

Pour cette SAE, nous étions en groupe de deux. Nous devions réaliser deux scripts Python en utilisant la librairie Scapy.

  • Script 1 (Attaque) : Nous devions pouvoir nous introduire dans une communication entre deux machines au niveau 2 du modèle OSI. L'objectif était de corrompre le cache ARP d'une cible (ARP Poisoning / Man in the Middle).
  • Script 2 (Découverte) : Nous devions réaliser un scanner réseau. Découverte active (ICMP avec commande "-a"), découverte passive (écoute ARP avec commande "-p"). Il devait être possible de scanner une plage complète avec "-t" et d'exporter les résultats dans un fichier avec "-x".

SAE 3.01 : Prise Connectée (IoT)

Présentation

Réalisation d'une prise connectée pilotable à distance via un site web et une application.

Explication & Réalisation

Pour cette SAE, nous étions en groupes de quatre. L'objectif était de domotiser une prise électrique.

Matériel utilisé : Raspberry Pi, Arduino UNO, Capteur de température DS18B20.

Architecture technique :

  • Mise en place d'un serveur Mosquitto (MQTT) sur le Raspberry Pi pour communiquer avec l'Arduino.
  • Développement d'une page web sécurisée (Login/Mot de passe).
  • Synchronisation en temps réel entre la page web et l'application mobile de pilotage.

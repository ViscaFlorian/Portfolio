SAE 3.03 : Architecture Multisite

Présentation

Conception d'une maquette de réseau d'entreprise multisite (e-commerce) pour réaliser leur interconnexion sécurisée.

Explication & Réalisation

Nous étions en groupe de trois. Nous avons utilisé le simulateur GNS3. Le projet était divisé en trois grandes parties techniques :

1. Partie Services

Mise en place de l'infrastructure serveur :

  • DNS Primaire : Windows Server 2019.
  • DNS Secondaire : Machine Linux (CLI).
  • Web : Serveur Apache hébergeant un Intranet et un site externe.
  • Mail : Serveur SMTP fonctionnel.

2. Partie LAN

Conception d'une infrastructure locale adaptée à la taille de l'entreprise (VLANs, commutation, redondance).

3. Partie FAI (Interconnexion)

Mise en place du réseau opérateur pour permettre la communication entre les différents sites géographiques de la société.

Aperçu de la topologie réalisée sous GNS3

SAE 1.04 : Se présenter en ligne

Présentation

Nous avons dû réaliser un site web statique.

Explication & Réalisation

Pour cette SAE, nous étions seuls. Nous avons dû réaliser un site web statique. Ce site web devait avoir deux pages. Le premier devait être une page de présentation, il devait avoir notre parcours effectué jusqu'à présent et les projets que nous voulons faire plus tard.

Dans la deuxième page, nous devions présenter un centre d'intérêt. Pour finir, nous devions pouvoir traduire notre site web en anglais. En plus de ce travail, nous avons dû réaliser une recherche sur nous-même pour connaître notre e-réputation, nous devions expliquer ce qui était bien ou mal sur celle-ci.

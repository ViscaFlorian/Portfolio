SAE 2.03 : Solution Web & BDD

Présentation

Pour cette SAE, nous avons dû réaliser un site web dynamique avec séparation de la base de données et de la publication.

Explication & Réalisation

Pour cette SAE, nous étions en groupes de trois. Nous avions pour objectif de réaliser un site web dynamique sur le thème de la filmographie (Django).

  • Fonctionnalités : Ajout de films, d'acteurs, gestion d'utilisateurs (amateurs vs professionnels) et système de commentaires différenciés.
  • Architecture : Contrairement aux TP classiques, nous devions utiliser une base de données MySQL externe (pas celle de Django par défaut) et publier le site web via un autre moyen que le serveur de développement Django.
